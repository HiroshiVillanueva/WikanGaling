Features of our Website:

Welcome to the features panel! To start off, all these right here are what you need to get started with to finally make your first Module!

Whether it be a quiz, an exam, or even a simple exercise! all these question types are open for any elementary Teacher to use as they please:

  • ◉ The multiple-choice question format is a type of assessment where a question is followed by several possible answers. The task is to select the correct option from the given choices.

  • ◉ The matching type question format presents two columns: one with prompts and the other with possible answers. The task is to correctly pair each item from the first column with its corresponding match in the second.

  • ◉ The fill-in-the-blanks question format provides a sentence with a missing word or phrase. Students must supply the correct answer to complete the statement.

These are all the current available question formats for our module editor!
