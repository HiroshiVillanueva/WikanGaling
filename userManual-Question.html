Core Features

1. Multiple Choice Format

a type of assessment where a question is followed by several possible answers, and the student must select the correct one from the given options. It’s commonly used to test knowledge, understanding, or decision-making skills efficiently.

2. Matching type Format

consists of two columns: one with questions or prompts and the other with possible answers. The student’s task is to correctly pair each item in the first column with its corresponding match in the second column.

3. Sentence Builder Format

presents a sentence or statement with one or more missing words, which the student must complete by writing the correct answer in the blank space. This type of question tests recall and understanding of key concepts.
