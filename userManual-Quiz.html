USER-MANUAL

Core Features

1. Student Landing Page:

Welcomes new users and requires a quick sign-in. Students should type their full names into the input field and then select the 'Ipagpatuloy' button to continue.

2. Module Landing Page:

Clearly displays the module title and a brief description. Students can begin the module by clicking the 'SIMULAN' button.

3. Questions Interface:

The module title is always displayed at the top for context, and students can track their progress using a visual progress bar and an accompanying label (e.g Tanong 1 ng 6) located nearby.

4. Multiple Choice Interface:

The main content area displays the question itself, followed by a set of options from which the student must select their answer.

The system provides immediate feedback upon answer selection. As soon as the student chooses an option, the selected option itself is highlighted in green if it is correct, or highlighted in red if it is incorrect. A corresponding message appears confirming the correct or incorrect status of the selected answer.

5. Matching Type Interface:

The main content area is set up with two distinct columns. These columns contain the items that need to be paired, which may consist of text, images, or a combination of both.


To answer the question, the student simply clicks on an item in the first column and then clicks on its corresponding match in the second column to establish the connection.


The system provides immediate visual feedback: correct pairings are instantly connected by a solid green line, while incorrect pairings show a broken red line, and a matching message appears immediately to confirm the outcome. Once all items are matched, the student selects the 'SUSUNOD' (Next) button to move on to the next question.

6. Sentence Builder Interface:

The main area presents a partially completed sentence or statement that requires an answer.


Students must use the drag-and-drop method to answer: they drag an option from the list of available choices and drop it into the target box labeled 'Ihulog Dito'. Once the blank is filled, the student selects the 'SUSUNOD' (Next) button to move on to the next question.

The system provides immediate feedback upon the successful action: the selected option is highlighted in green if correct, or highlighted in red if incorrect. If the answer is wrong, the correct answer is immediately displayed at the side for reference.

7. Quiz Results Screen:

Upon completing the module, the student is taken to the Quiz Results Screen. This page features a festive celebratory message and a congratulatory note. The student's final performance is highlighted by their score, which is clearly displayed as a fraction, such as 3 / 6.


At the bottom, students are presented with three action buttons: 'ULITIN' to retake the module which would redirect to the Module Landing Page, 'KABUUAN' to view a detailed breakdown summary of the students answers for each question, and 'TAPUSIN' to exit the activity and redirect to the Student Landing Page.
