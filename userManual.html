Introduction

WikanGaling began as a school project created by the three of us — classmates who loved language and interactive design. We noticed younger students zoning out during language drills but lighting up for games. Our goal was simple: make language learning feel like play. This manual will walk you through every feature we built so teachers, parents, and students can get the most out of LinguaQuest.

Core Features (what we made up)

1. Classroom Mode (for teachers)

Teachers can create a timed session where every student joins with a code. The teacher controls which mini-games run and can broadcast quick polls or pronunciation tasks live.

2. Parent & Teacher Dashboard

Simple visual reports show weekly progress, mastery badges, streaks, and suggestions for offline practice. You can set reminders or enable weekly printable worksheets generated from the child's recent lessons.

3. Custom Lesson Builder (teacher-friendly)

Drag-and-drop builder where teachers or parents create 5–10 minute lesson blocks by mixing vocabulary lists, images, voice prompts, and a mini-game. Each custom lesson can be shared via a short code.

5. Accessibility & Multilingual Support

Read-aloud for UI text, adjustable text size, high-contrast mode, and hint-level control. Languages available in our fictional launch: English, Spanish, Filipino, and Mandarin (sample content only).

Getting Started — Step by step

Step 1 — Create an account

We recommend signing up with a teacher or parent account first. Click Sign up, select your role (Teacher / Parent / Student), then follow the prompts.

Step 2 — Create learner profiles

For each child, create a profile: name, target language, approximate age (helps difficulty), and avatar base. You can create multiple profiles under one parent account.

Step 3 — Choose a learning path

Pick from themed paths like First Words, My Classroom, or Animals & Nature. Each path maps to 6 weeks of quests and games.
